run_test(self, testfile=None, times=1)

source code 

Performs a test. The test is generated from a given file or randomly. To better check for synchronization errors the test should be run several times, as given by the 'times' parameter.

Parameters:
  • testfile (String) - the path+name of the file containing the test parameters
  • times (Integer) - number of time to run the test

start_test(self, test)

source code 

Starts the execution of a given test. It generates the nodes and creates threads that send tasks to the cluster's nodes and wait for the result.

Parameters:
  • test (Test) - a Test object containing the test's parameters (nodes, matrix, tasks etc)
